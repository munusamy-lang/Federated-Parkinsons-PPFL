# Federated-Parkinsons-PPFL

Federated Learning + Blockchain + Homomorphic Encryption demo implementation for Parkinson’s Disease prediction.

### Structure
- preprocessing/ — data cleaning & normalization
- federated_learning/ — FL server, clients, FedAvg
- encryption/ — simple additive masking (demo HE)
- blockchain/ — mock blockchain logger
- evaluation/ — accuracy, precision, recall metrics
